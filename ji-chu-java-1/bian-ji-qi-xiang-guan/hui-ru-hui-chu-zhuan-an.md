# 匯入匯出專案

## 匯入專案

1. Eclipse選單中 File &gt; Import...
2. General &gt; Existing Projects into Workspace
3.  選擇Select root directory 點選**Browse...**按鈕開啟專案的位置。 Projects勾選要開啟的專案，完成！
